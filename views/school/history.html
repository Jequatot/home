(Evil) Hangman

Fall 2016

a C++ game of Hangman, for my C++ class, which pulled from a dictionary text file
featured an "evil" mode, where whenever a letter was guessed, the game change its answer to one that did not contain that letter

for example, if the board read "oa--", and the player entered 't', the program would scroll through the dictionary for a four-letter word that began with "oa" but did not contain a 't', so it might come back and say "there are no 't's, the answer was 'oars'"
if you guessed 's', it would tell you the answer was "oath"
if you guessed 'h', it would tell you the answer was "oats"
